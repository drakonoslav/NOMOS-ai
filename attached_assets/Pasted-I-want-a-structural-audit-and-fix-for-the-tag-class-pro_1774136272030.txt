- quantified entity extraction
- entity category/tag assignment
- graph node creation
- graph query filtering
- graph constraint execution

3. Provenance loss
- graph node has tags, but there is no record of where they came from

- category, tag, and label are being used interchangeably

- graph queries use label substring matching where canonical tags should be used
